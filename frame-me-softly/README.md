# Love Photobooth 📸✨

A romantic vintage-inspired digital photobooth built with React, Framer Motion, and TailwindCSS.
Capture aesthetic photo strips with cinematic filters, ambient music, elegant animations, and downloadable memories.

---

## 🌹 Features

* 📷 Real-time webcam capture
* 🎞 Vintage & cinematic photo filters
* ✨ Animated flash and countdown effects
* 🎵 Built-in romantic playlist player
* 🖼 Downloadable photo strip PNG
* 💌 Elegant romantic UI/UX
* 📱 Responsive layout
* 🎨 Vintage photobooth aesthetic

---

## 🛠 Tech Stack

* React
* Vite
* TailwindCSS
* Framer Motion
* html2canvas
* Lucide React

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/mapuinblue/FrameMeSoftly.git   
```

Navigate into the project:

```bash
cd frame-me-softly
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

## 📸 How It Works

1. Allow camera permissions
2. Select a vintage filter
3. Press **Capture**
4. Complete all 3 shots
5. Download your photostrip as PNG
